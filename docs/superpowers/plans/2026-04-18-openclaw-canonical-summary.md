# OpenClaw Canonical Summary Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make canonical `summary.md` generation always use the real OpenClaw LLM path and remove the fake and legacy summary implementation.

**Architecture:** Route both stop-time canonical summaries and manual re-runs through `TranscriptPromptService.rewrite_canonical_summary()`. Remove the standalone `Summarizer` abstraction and the fake OpenClaw summary branch so the only remaining summary path is the OpenClaw-backed LLM request. Keep the `summary_model` job field, but populate it from `OPENCLAW_SUMMARY_MODEL` rather than a separate summary configuration surface.

**Tech Stack:** Python 3.14, FastAPI, Pydantic, SQLAlchemy, httpx, pytest

---

## File Structure

- `controller/app/jobs.py`
  Responsibility: stop-flow orchestration; switch canonical summary generation from `Summarizer` to `TranscriptPromptService`.

  Responsibility: published runtime contract; remove dead `SUMMARY_*` variables.

```python
def test_stop_uses_transcript_prompts_for_canonical_summary(client, monkeypatch) -> None:
    login(client)
    runtime = client.app.state.runtime
    start = client.post("/api/jobs/start", json={"title": "Authorized public livestream"})
    assert start.status_code == 200
    job_id = start.json()["id"]

    called: dict[str, str] = {}

    async def fake_rewrite_canonical_summary(*, job_id: str, prompt: str = "") -> str:
        called["job_id"] = job_id
        summary_path = runtime.artifacts.job_paths(job_id).summary
        summary_path.write_text("# Summary\n\nGenerated by OpenClaw\n", encoding="utf-8")
        return str(summary_path)

    def fail_legacy_summarize(**kwargs):
        raise AssertionError("legacy summarizer should not be used during stop")

    monkeypatch.setattr(runtime.transcript_prompts, "rewrite_canonical_summary", fake_rewrite_canonical_summary)
    monkeypatch.setattr(runtime.capture.summarizer, "summarize", fail_legacy_summarize)

    stop = client.post(f"/api/jobs/{job_id}/stop")

    assert stop.status_code == 200
    assert called["job_id"] == job_id
    assert runtime.artifacts.job_paths(job_id).summary.read_text(encoding="utf-8") == "# Summary\n\nGenerated by OpenClaw\n"
```
