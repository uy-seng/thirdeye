# Superpowers Notes

The files in this directory are archival planning and implementation notes from earlier development sessions. They are not the current public operator or runtime contract.

Some archived plans mention removed paths such as `controller/web` or `apps/web`, old login/session UI, runtime fake-mode helpers, fake summary providers, or legacy health-dashboard ideas. Treat those references as historical context only. Current behavior is documented in the top-level `README.md`, `SECURITY.md`, `docs/api.md`, `docs/architecture.md`, and `docs/operations.md`.
